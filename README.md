# Money Manager App

## 📌 Objective
This project is a web-based Money Manager application built using HTML, CSS, and JavaScript. It helps users manage income and expenses.

## 🚀 Features
- Add, Edit, Delete Transactions
- View Financial Summary (Income, Expense, Balance)
- Data stored using LocalStorage
- Form validation implemented
- Responsive UI

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- DOM Manipulation
- Event Handling
- Local Storage

## ⚙️ Technologies Used
- HTML
- CSS
- JavaScript

## ⚠️ Challenges Faced
- Handling edit functionality
- Updating UI dynamically
- Managing localStorage data

## 📚 Learnings
- Practical implementation of JavaScript
- OOP concepts
- Working with browser storage
